# 👩‍💻 Il Mio Portfolio Professionale - Sony

Benvenuti nel repository del mio portfolio personale! Sono una professionista con un background solido in ambito amministrativo e contabile a Venezia e Mestre, che ha deciso di unire la precisione analitica alla passione per le tecnologie web.

## 🚀 Obiettivo del Progetto
Questo sito nasce con lo scopo di mettere in pratica le mie competenze nello sviluppo Front-End e di mostrare la mia evoluzione nel mondo del codice. È il mio spazio creativo digitale, ottimizzato per essere chiaro, moderno e accessibile.

## 🛠️ Tecnologie Utilizzate
* **HTML5:** Struttura semantica, pulita ed efficace;
* **CSS3:** Stile moderno, cura del dettaglio visivo e layout responsive per smartphone;
* **JavaScript:** Logica d'interazione per elementi dinamici (selettore della lingua e pulsanti).

## 📈 Il mio percorso
Sto studiando con dedizione per padroneggiare le logiche del web e trasformare questa passione in un'opportunità lavorativa concreta. Ogni riga di codice di questo portfolio rappresenta un passo avanti verso il mio obiettivo.
